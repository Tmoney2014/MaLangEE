# 기술 스택 상세

## Frontend (주요 개발 영역)
- **Framework**: Next.js 16.1.0 + React 19.2.3
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **State Management**: 
  - TanStack React Query v5 (서버 상태)
  - Zustand v5 (클라이언트 상태)
- **Form**: React Hook Form 7 + Zod 4
- **Testing**: 
  - Vitest (단위 테스트)
  - Playwright (E2E 테스트)
- **Package Manager**: yarn
- **Node Version**: 20.x (`.nvmrc` 파일 참조)

## Backend
- **Framework**: FastAPI
- **AI Integration**: OpenAI Realtime API
- **WebSocket**: 실시간 음성 스트리밍

## Database
- **DBMS**: PostgreSQL 15+
- **Database Name**: malangee
- **Port**: 5432

## 개발 도구
- **ESLint**: Next.js Core Web Vitals 설정
- **Prettier**: Tailwind CSS 플러그인 포함
- **Storybook**: 컴포넌트 문서화 (포트 6006)
